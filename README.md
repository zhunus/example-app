## Локальная установка

```bash

npm install

npx auth

```

Необходимо создать файл `.env.local` и скопировать в него содержимое `.env` файла подставив необходимые значения

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
